# reference_review_diy

## 概述

**论据-引文-评判**审核技能。对学术专著、论文、报告等长文进行系统性的引文审核——提取关键观点/数据主张，验证引用文献的真实性，评估引文对主张的支持程度。

## 功能

| 功能 | 说明 |
|------|------|
| 论据提取 | 自动识别正文中的关键数据主张和论点 |
| 引文验证 | WebSearch验证每条参考文献是否真实存在 |
| 支持度评判 | ★★★★★（完全支持）→ ★（不支持）五级评判 |
| 多Agent并行 | 大型文档分篇章并行审核 |
| 结构化报告 | 输出论据-引文-评判对照表+问题汇总+修正建议 |

## 触发方式

用户说以下任意一句即加载：
- "审核参考文献" / "审核引文"
- "论据-引文-评判"
- "检查引文真实性"
- "验证参考文献是否支持观点"

## 评判标度

| 评级 | 含义 |
|:---:|------|
| ★★★★★ | 引文直接包含正文主张的精确数据或结论 |
| ★★★★ | 引文整体方向一致，但有微小差异 |
| ★★★ | 部分支持，但不是主张的主要来源 |
| ★★ | 主题相关，但不直接支持具体主张 |
| ★ | 引文与主张明显无关 |

## 应用案例

2026年7月，此技能首次应用于《神经干细胞与脑损伤修复：从基础到临床》专著（约43.8万字，14章）的全书引文审核。三Agent并行，从约190条核心论据中发现3处紧急错配、约10处中低优先级问题。

## 依赖

- 无外部Python依赖
- 需要WebSearch能力的Agent环境
- 建议大文档使用多Agent并行模式
